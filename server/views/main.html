Instagram Twitter Preview

Create a twitter preview card for any instagram post.

  1. Find a public image in instagram. Click on the image and then copy the url (something like https://www.instagram.com/p/B5F0TjuHAsn/) into the box below.
  2. Click the button to create a shareable link
  3. Paste link into tweet

Link copied to clipboard! Now go paste it in twitter
